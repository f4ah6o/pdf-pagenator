# PDF ページ番号付与 SPA — 実装指針（フロントエンドのみ / GitHub Pages 配備）

## 要件
- PDFファイルにページ番号を追記するSPA（シングルページアプリ）
- **GitHub Pages** にデプロイ可能な **静的サイト**
- **完全サーバーレス**（PDFはアップロードせず、ブラウザで処理）
- データはユーザーのPCのブラウザ内（外部送信なし）
- ページ番号位置：ヘッダー or フッター
- 寄せ：左・中央・右
- **総ページ数を含めるかどうか**を選択可能

## 技術スタック
| 目的 | 使用技術 |
|---|---|
| SPA / UI | React + TypeScript + Vite |
| PDF編集 | `pdf-lib` |
| 配布・ホスティング | GitHub Pages |
| スタイル | 任意（Tailwind or CSS）|
